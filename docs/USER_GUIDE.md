# User Guide

## Getting Started
1. Open the app URL or install the APK.
2. Allow location permissions to find nearby bars.

## For Bartenders
- **Joining**: Select your bar. Choose "Bartender" role.
- **Sending Requests**: Tap the button corresponding to your need (e.g., "ICE").
- **Monitoring**: You will see a spinner or indicator that the request is sent.
- **Completion**: You will be notified when a Barback claims or completes the task.

## For Barbacks
- **Joining**: Select your bar. Choose "Barback" role.
- **Receiving Requests**: Keep the app open (or backgrounded if notifications are enabled).
- **Claiming**: Tap an "Open" request to claim it. This lets others know you are on it.
- **Completing**: Tap "Done" when the task is finished.

## Troubleshooting
- **No Sound?**: Check the "Bell" icon in the corner. Ensure system volume is up.
- **Not Syncing?**: Check your internet connection. The app requires data/WiFi.
